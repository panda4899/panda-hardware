Panda-Hardware
======

Devices, layouts, schematics, and parts for Team PANDA. 

Raspberry Pi B+ GPIO Header 2x20 pin, RJ45-8, LM358, M03, M06, M07, and M09 from [Sparkfun](https://github.com/sparkfun/SparkFun-Eagle-Libraries)
